    üéØ 3. Reading Strategy for Announcements¬∂

    Part 6: Text Completion Strategy¬∂

    üìù STEP-BY-STEP APPROACH:

1. QUICK SCAN (15 seconds)
   - Identify text type (memo, notice, announcement)
   - Note sender, recipient, subject
   - Understand general purpose

2. ANALYZE EACH BLANK (30 seconds per blank)
   - Read sentence before the blank
   - Read sentence after the blank  
   - Consider grammar and logic
   - Check answer choices

3. CONTEXT VERIFICATION (10 seconds)
   - Does answer fit logically?
   - Is grammar correct?
   - Does it match tone/style?

    Part 7: Comprehension Questions Strategy¬∂

    üîç QUESTION-FIRST APPROACH:

1. READ QUESTIONS FIRST (30 seconds)
   - Identify what information to find
   - Note keywords to scan for
   - Understand question types

2. STRATEGIC SCANNING (45 seconds)
   - Locate relevant sections quickly
   - Focus on dates, names, locations
   - Look for cause-effect relationships

3. ANSWER WITH EVIDENCE (30 seconds per question)
   - Find specific text support
   - Eliminate wrong answers
   - Choose best match

    Common Question Types:¬∂

    Question Pattern Strategy Example
    Purpose questions Look for main announcement reason "Why was this notice issued?"
    Detail questions Scan for specific information "When does the new policy take effect?"
    Inference questions Read between the lines "What can be inferred about the company?"
    Vocabulary questions Use context clues "What does 'comply' mean in this context?"
